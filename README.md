# AutoShow

This is a sample single-page application for an automobile showroom. It allows users to:

- View a list of available vehicles
- Filter vehicles by type (sedan, SUV, truck, electric)
- Sort vehicles by price and year
- Search for vehicles by make, model, or year
- View vehicle details in a modal window
- Show favorite vehicles
- Contact sales via a form

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Since this is a simple HTML file with embedded CSS and JavaScript, you can run it by simply opening the `index.html` file in your web browser. No special setup is required.
